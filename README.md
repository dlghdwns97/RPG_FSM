# 3D 플랫포머 점프 게임

유니티 심화 강의 자료를 토대로 만들어본 플랫포머 형식의 3D 게임 입니다. 플레이어는 정해진 오브젝트 위에서 떨어지지 않고 점프로 결승선의 초록 구체에 다가가야 합니다.
